# skills CLI 1.5.17 Integration Spike

## Outcome

Use an external CLI process behind `SkillCommandGateway`; do not import the package as a JavaScript library. Pin `skills@1.5.17` and invoke it as:

```text
npm exec --yes --package=skills@1.5.17 -- skills <args>
```

The package publishes CLI binaries but no documented stable library export. UI code must not parse or depend directly on its human-oriented output.

## Verified safe contract

An isolated run passed exact version detection, root command discovery, JSON listing, local source discovery, project install, inventory after install, one-time use, project removal, and unknown-command diagnostics.

Evidence: [passing timeout-guarded results](runs/20260715-134100/results.json).

## Behavioral hazards found

- The shorter `npx skills@1.5.17` form was ambiguous in this environment; `npm exec --package` reliably selected the pinned binary.
- Subcommand `--help` is not consistently help-only.
- `update --help` executed a real update rather than showing help.
- `init --help` created a skill named `--help`.
- An unknown command printed an error but returned exit code zero.
- A UTF-8 BOM before `SKILL.md` frontmatter prevented fixture discovery.

Therefore, AMS must verify semantic outcomes and filesystem state rather than trusting exit codes or help conventions.

## Incident during exploratory probing

Before the isolated runner existed, probing `update --help` unexpectedly updated two user-global skills: `find-skills` and `agent-browser`. No rollback was attempted because there was no pre-operation snapshot and guessing previous content would be unsafe. Probing `init --help` also created `--help/SKILL.md` in this workspace; that artifact was identified and removed.

This incident is the reason all repeatable mutation probes now run with isolated `HOME`, `USERPROFILE`, `CODEX_HOME`, project directory, and npm cache.

## Product error handling

The gateway maps failures to stable categories: runner unavailable, version mismatch, timeout, network failure, invocation failure, unparsable output, contract change, and unexpected write. The UI offers retry, diagnostics, copy issue report, or cancel. It never silently upgrades the pin or retries a mutation.

## Repeatable workflow

Use the project-scoped [`spike-skills-cli`](../../../.agents/skills/spike-skills-cli/SKILL.md) skill. Its runner creates a temporary fixture and sandbox, restores process environment variables, writes a JSON report, and removes only a verified child of its dedicated temp root.
